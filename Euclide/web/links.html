Dynamic geometry links

Here are some applications which have a quite similar goal. Graphical applications, command-text programs, and geometry libraries are differentiated.

Dynamic geometry software

Sofware to do dynamic geometry with a user-friendly interface:

  • Cabri, one of the references
    http://www-cabri.imag.fr/
  • Cindirella: another reference
    http://www.maths.univ-rennes1.fr/~coste/cindy/menucindy.html
  • LiveGraphics: a Java applet for 3D geometry
    http://wwwvis.informatik.uni-stuttgart.de/~kraus/LiveGraphics3D/index.html
  • DrGEO, another software of interactive geometry
    http://ofset.sourceforge.net/drgeo/index-fr.html
  • KGeo is a version of DrGEO for Linux.
    http://kgeo.sourceforge.net/

Geometry tools

Some software for people how do not mind use a text commands to obtain results:

  • Eukleides: a LaTeX package to draw geometric figures. Seems interesting.
    http://eukleides.free.fr/
  • GCLC (for Geometry construction LaTeX converter): anoter interesting soft, for LaTeX users.
    http://www.matf.bg.ac.yu/~janicic/gclc/

Geometry Library

Developers may wish toolboxes for geometric computations

  • javaGeom: the library used by Euclide. Also in development...
    http://geom-java.sourceforge.net/
  • LiveGraphics: a Java applet for 3D geometry.
    http://wwwvis.informatik.uni-stuttgart.de/~kraus/LiveGraphics3D/index.html

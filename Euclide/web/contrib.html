Contribute to Euclide

If you enjoy the software, why not help making it better ? You can bring your contribution in several ways.

Bug reports and feature request

If the functionality you need is laking, please report it. There is a bug tracker and a feature request tracker. They only ask to be populated !

Translations

Euclide is currently localized in english and in french. It is possible to translate to any other language by adding the appropriate property file.

Please contact the author (dlegland at yahoo dot fr) if you are interested.

Coding

The software itself is open-source. If you feel motivated enough, you can grab the last version of the development repository, modify it, and sent the modifications to the author (dlegland at yahoo dot fr).

To get the latest repository version:

svn co https://jeuclide.svn.sourceforge.net/svnroot/jeuclide/trunk euclide
